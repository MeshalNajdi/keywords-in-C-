## Keyword `Class`
 Class  تستخدم لتجميع عناصر تشترك في عدة خصائص , ويمكن له ان يرث خصائص من "كلاس" اخر .
** مثال :**

<div dir = ltr > 

{


    public class People { 
    string name ; 
    int age ;
    public people() {
        this.name = "name";
        this.age = 20 ;
    }
    }
    public class students : people {

        public student () : Base () {
            Console.WriteLine("this is a student")
        }
    }
}
