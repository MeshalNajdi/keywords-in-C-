## Keyword `continue`
 
أمر يستخدم للإنهاء والخروج من اللفة المتواجده فيها العبارة .

**مثال :**

<div dir = ltr > 

{


   for (int i = 1; i < 10; i++) عبارة تكرار
        {
            if (i == 5)  اذا تحقق الشرط 
            {
                continue ; سيتم الخروج من اللفه بدون طباعه الرقم ولكن سيستمر بالتكرار
            }
            Console.WriteLine(i);
        }

}
