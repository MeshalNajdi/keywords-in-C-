## Keyword `public`
 هي إحدى كلمات تحديد الوصول ولكنها الاقل تحديدا للخصوصية حيث انها تسمح للجميع بالوصول للبيانات والدوال .

**مثال :**
 في المثال ادناه نلاحظ ان الكلمه "public "  تجعل جميع بيانات النوع "people " قابلة للوصول من الجميع . 

<div dir = ltr > 

{


    public class People  
     { 
    public string name ; 
    public int age ;
    public people() {
        this.name = "name";
        this.age = 20 ;
    }
   
}
