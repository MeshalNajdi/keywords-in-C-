## Keyword `Base`
Base كلمة تستخدم من كلاس تابع للوصول لعناصر الكلاس الاساسي ، من الممكن ان تستخدم لاستدعاء دالة او باني من الكلاس الاساسي .
** مثال :**

<div dir = ltr > 

{


    public class People { 
    string name ; 
    int age ;
    public people() {
        this.name = "name";
        this.age = 20 ;
    }
    }
    public class students : people {

        public student () : Base () {
            Console.WriteLine("this is a student")
        }
    }
}
