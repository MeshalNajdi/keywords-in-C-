## Keyword `enum`
 تستخدم دائما لتعريف الثوابت المرتبطة ببعضها بطريقة ما .


**مثال :**

 في المثال ادناه نلاحظ ان الكلمه "enum " استخدمت لتعريف ايام الاسبوع
<div dir = ltr > 

{


    enum weekdays {
    sunday,
    monday,
    tuesday,
    wednesday,
    thursday,
    friday }
   
    
   
}
