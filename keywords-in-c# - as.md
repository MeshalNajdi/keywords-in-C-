## Keyword `as`
as كلمة تستخدم لتحويل نتيجة عبارة بشكل صريح إلى متغير من نوع " رفرنس " أو  لا تؤشر على قيمة ان كانت لا تستطيع التحويل الى القيمة المطوبة  .
** مثال :**

<div dir = ltr > 

{


    public class People { 
    string name ; 
    int age ;
    public people() {
        this.name = "name";
        this.age = 20 ;
    }
    }
    public class students : people {

        public student () : Base () {
            Console.WriteLine("this is a student")
        }
    }
}
